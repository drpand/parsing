## [1.0.0] — 2026-03-15

### ✨ Новое
- **AI парсинг через OpenRouter** — Vision API для анализа скриншотов товаров
- **Автопостинг в Telegram канал** — планировщик APScheduler (интервал 15 мин)
- **Админ-панель** — полное управление товарами и настройками
- **Кэширование результатов** — 24 часа (экономия API запросов)
- **Кнопка связи с менеджером** — с предзаполненным сообщением (артикул + название)
- **Process Manager** — автозавершение предыдущего экземпляра при запуске
- **PID файл** — корректное управление процессом

### 🔧 Исправления
- **Постинг** — 1 фото + 1 кнопка (убраны дубли)
- **Артикул** — передаётся в кнопке через URL encoding
- **Редактирование** — исправлена ошибка после поля "В наличии"
- **Ссылка на товар** — полная (не обрезается)
- **Selenium** — улучшен Lazy Scrolling (до 5 прокруток с детекцией конца)
- **БД** — исправлена работа с сессиями (`async with database.get_session()`)

### 📦 Зависимости
- **Добавлены:** `selenium`, `selenium-stealth`, `webdriver-manager`
- **Удалены:** `playwright`, `playwright-stealth`

### 🗄️ База данных
- **Миграция:** добавлена колонка `last_posted_at` в таблицу `products`

### 📁 Файлы
- `app/core/version.py` — управление версией
- `app/core/process_manager.py` — PID менеджер
- `app/services/scheduler/fx_scheduler.py` — FX планировщик (готов к использованию)
- `app/services/proxy/proxy_manager.py` — Proxy менеджер (готов к использованию)
- `app/utils/formatters/formatters.py` — форматтеры цен и текста
