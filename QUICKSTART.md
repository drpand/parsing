# ⚡ IndiaShop Bot v1.2.0 — Быстрый старт (5 минут)

**Для кого:** Новые пользователи, которые хотят быстро запустить бота  
**Время:** 5-7 минут

---

## 🎯 ЧЕКЛИСТ ЗАПУСКА

Пройдите все шаги по порядку:

### [ ] Шаг 1: Python установлен?

```bash
python --version
```

Если видите `Python 3.11.x` или выше → ✅ Пропускайте

Если ошибка → Скачайте с https://www.python.org/downloads/

⚠️ **Важно:** При установке отметьте ✅ "Add Python to PATH"

---

### [ ] Шаг 2: Запустите SETUP.bat

1. Откройте папку с ботом
2. Дважды кликните `SETUP.bat`
3. Дождитесь: `✅ НАСТРОЙКА ЗАВЕРШЕНА!`

---

### [ ] Шаг 3: Получите BOT_TOKEN (3 мин)

1. Откройте Telegram
2. Найдите **@BotFather**
3. Напишите `/newbot`
4. Придумайте имя: `MyIndiaShop`
5. Придумайте username: `MyShopTest_bot`
6. Скопируйте токен: `123456789:ABCdef...`

---

### [ ] Шаг 4: Получите OPENROUTER_API_KEY (5 мин)

1. Откройте https://openrouter.ai/keys
2. Войдите через Google/GitHub
3. Нажмите **"Create Key"**
4. Название: `IndiaShop`
5. Скопируйте ключ: `sk-or-v1-...`

⚠️ **Копируйте сразу!** Ключ показывается 1 раз

---

### [ ] Шаг 5: Узнайте свой Telegram ID (1 мин)

1. Откройте Telegram
2. Найдите **@userinfobot**
3. Напишите `/start`
4. Скопируйте ID: `123456789`

---

### [ ] Шаг 6: Заполните .env

1. Откройте файл `.env` (в папке с ботом)
2. Найдите строки:

```env
BOT_TOKEN=your_token_here
OPENROUTER_API_KEY=your_api_key_here
ADMIN_TELEGRAM_IDS=your_telegram_id_here
```

3. Вставьте значения:

```env
BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
OPENROUTER_API_KEY=sk-or-v1-abcdef1234567890...
ADMIN_TELEGRAM_IDS=123456789
```

4. **Сохраните** (Ctrl+S)

---

### [ ] Шаг 7: Запустите бота

1. Дважды кликните `START.bat`
2. Откроется чёрное окно (консоль)
3. Дождитесь: `✅ Bot is ready!`

⚠️ **Не закрывайте окно!** Бот работает пока окно открыто

---

### [ ] Шаг 8: Проверьте работу

1. Откройте Telegram
2. Найдите своего бота (по username из Шага 3)
3. Напишите `/start`
4. Бот должен ответить меню

✅ **ЕСЛИ МЕНЮ ПОЯВИЛОСЬ — ГОТОВО!**

---

## 🎉 ЧТО ДАЛЬШЕ?

### Быстрые команды:

| Команда | Описание |
|---------|----------|
| `/admin` | Админ-панель |
| `/start` | Главное меню |
| `/version` | Версия бота |

### Первый парсинг:

1. `/admin` → **📦 Добавить товар**
2. Отправьте URL товара с Zara/Myntra
3. Дождитесь превью
4. Нажмите **✅ Подтвердить**

### Первая публикация:

1. `/admin` → **📦 Товары**
2. Выберите товар
3. Нажмите **📤 Опубликовать**
4. Выберите канал

---

## 🐛 ЧТО-ТО ПОШЛО НЕ ТАК?

### "BOT_TOKEN не найден"

→ Откройте `.env` и вставьте токен из @BotFather

### "Доступ запрещён"

→ Проверьте `ADMIN_TELEGRAM_IDS` в `.env` (ваш ID из @userinfobot)

### Бот не отвечает

1. Закройте окно бота
2. Проверьте `.env`
3. Запустите `START.bat` снова

### Нужна помощь?

→ Напишите **@tatastu** в Telegram
