## ⚡ БЫСТРЫЙ СТАРТ (5 минут)

### Шаг 1: Распакуйте архив

Распакуйте папку `IndiaShop_Bot_v1.2.0` в удобное место, например:
```
C:\Bots\IndiaShop_Bot_v1.2.0\
```

⚠️ **Важно:** Путь не должен содержать кириллицу или пробелы!

### Шаг 2: Запустите настройку

1. Откройте папку с ботом
2. Дважды кликните **`SETUP.bat`**
3. Дождитесь завершения установки

### Шаг 3: Получите API ключи

Следуйте инструкции в файле **`INSTALL_GUIDE.md`**:
1. **BOT_TOKEN** — через @BotFather (3 мин)
2. **OPENROUTER_API_KEY** — на openrouter.ai (5 мин)
3. **ADMIN_TELEGRAM_IDS** — через @userinfobot (1 мин)

### Шаг 4: Настройте .env

1. Откройте файл `.env` (в той же папке)
2. Вставьте полученные ключи:

```env
BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrsTUVwxyz
OPENROUTER_API_KEY=sk-or-v1-abcdef1234567890...
ADMIN_TELEGRAM_IDS=123456789
```

3. Сохраните файл (Ctrl+S)

### Шаг 5: Запустите бота

1. Дважды кликните **`START.bat`**
2. Откроется консоль с логами бота
3. **Не закрывайте окно!** (бот работает пока окно открыто)

### Шаг 6: Проверьте работу

1. Откройте Telegram
2. Найдите своего бота (по username который создали в @BotFather)
3. Нажмите `/start`
4. Бот должен ответить меню

✅ **Готово!** Бот работает!

## 🎯 СЦЕНАРИИ ИСПОЛЬЗОВАНИЯ

### Сценарий 1: Парсинг товара с Zara

1. Админ: `/admin` → **📦 Добавить товар**
2. Отправьте URL товара: `https://www.zara.com/in/en/...-p12345.html`
3. Бот загрузит страницу, сделает скриншот
4. AI распознает: название, цену, наличие
5. Бот покажет превью
6. Админ подтверждает → товар в базе

### Сценарий 2: Парсинг категории

1. Админ: `/admin` → **📊 Парсинг категории**
2. Выберите категорию: `men_shirts`, `women_dresses`, и т.д.
3. Укажите лимит: `10` товаров
4. Бот спарсит категорию через Selenium + AI
5. Товары сохранятся в базу

### Сценарий 3: Публикация в канал

1. Менеджер: **📦 Товары** → выберите товар
2. Нажмите **📤 Опубликовать**
3. Выберите канал: `@tatastutest`
4. Бот отправит фото + описание + цену

### Сценарий 4: Менеджер обрабатывает заказ

1. Клиент пишет менеджеру: "Хочу заказать товар #123"
2. Менеджер: `/find 123` → карточка товара
3. Менеджер отправляет клиенту фото и цену
4. Клиент подтверждает → менеджер оформляет заказ

## 🐛 ДИАГНОСТИКА ПРОБЛЕМ

### Бот не запускается

**Проверьте:**
1. Python установлен? → `python --version`
2. Зависимости установлены? → Запустите `SETUP.bat`
3. `.env` заполнен? → Проверьте BOT_TOKEN и OPENROUTER_API_KEY

### Бот не отвечает на команды

**Решение:**
1. Проверьте `ADMIN_TELEGRAM_IDS` в `.env`
2. Узнайте свой ID через @userinfobot
3. Перезапустите бота (`STOP.bat` → `START.bat`)

### Ошибка "OpenRouter API failed"

**Причины:**
- Неверный API ключ
- Закончился баланс на OpenRouter
- Неправильное название модели

**Решение:**
1. Проверьте ключ на https://openrouter.ai/keys
2. Пополните баланс (от $5)
3. Проверьте название модели в `.env`

### Selenium не загружает страницы

**Решение:**
1. Установите Chrome/Chromium
2. Проверьте что ChromeDriver в PATH
3. Попробуйте другой User-Agent в настройках
