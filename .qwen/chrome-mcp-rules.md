# Chrome MCP — Правила использования

## 🔐 ГЛАВНОЕ ПРАВИЛО

**Chrome MCP сессия должна оставаться ОТКРЫТОЙ постоянно!**

### ❌ НЕЛЬЗЯ:
- Закрывать браузер после каждого действия
- Перезапускать Chrome MCP
- Терять сессию

### ✅ НУЖНО:
- Использовать ОДНУ сессию для всех действий
- Сохранять авторизации (Telegram, Claude.ai, GitHub)
- Переключаться между вкладками вместо создания новых

---

## 📋 РАБОЧИЙ ПРОЦЕСС

1. **Первый запуск:**
   - `mcp__chrome__browser_navigate` → https://web.telegram.org/a/
   - Авторизоваться
   - Сессия сохранена

2. **Последующие действия:**
   - Использовать ту же сессию
   - Переключаться между вкладками если нужно
   - НЕ закрывать браузер

3. **Для Claude.ai:**
   - Открыть в новой вкладке
   - Авторизоваться
   - Оставить открытым

---

## 🎯 ПРИМЕРЫ

### ✅ Правильно:
```
1. browser_navigate → Telegram Web
2. browser_click → открыть чат
3. browser_evaluate → отправить сообщение
4. browser_navigate → Claude.ai (та же сессия)
5. browser_evaluate → отправить промпт
```

### ❌ Неправильно:
```
1. browser_navigate → Telegram
2. browser_close → ❌ СЕССИЯ ПОТЕРЯНА!
3. browser_navigate → Claude.ai → ❌ НУЖНО ЛОГИНИТЬСЯ СНОВА!
```
