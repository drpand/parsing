## 🚀 Как обновиться

### 1. Обновите файлы

```bash
git pull  # или скопируйте файлы вручную
```

### 2. Проверьте .env

Убедитесь что Supabase поля пустые (если не используете):

```env
SUPABASE_URL=
SUPABASE_KEY=
```

### 3. Перезапустите бота

```bash
# Остановите старого бота (Ctrl+C)
python main.py
```

### 4. Протестируйте каталог

```
🛍 Каталог → Выберите категорию
```

## 🐛 Известные проблемы

### Нет товаров в категориях

**Причина:** Старые товары имеют категории из v1.1 (`clothing`, `shoes`)

**Решение:** Запарсить товары заново через AI Vision

### AI возвращает старые категории

**Причина:** Кэширование старых результатов

**Решение:** Очистить кэш:
```
/cache → Очистить всё
```

## 📊 Миграция с v1.1.0

### Автоматическая

Бот совместим со старыми товарами. Категории `clothing`, `shoes` будут работать.

### Ручная (рекомендуется)

Для получения детальных категорий:

1. Удалить старые товары
2. Запарсить категорию заново через AI
3. AI присвоит точные подкатегории
